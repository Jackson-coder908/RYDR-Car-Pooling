# RYDR - Car Pooling System 🚗💨

**RYDR** is a modern, real-time carpooling application designed to connect drivers with empty seats to passengers traveling in the same direction. Built with **React** and **Vite**, it focuses on speed, ease of use, and efficient ride-sharing.

---

## 🚀 Features
* **User Authentication:** Secure login and signup for drivers and passengers.
* **Ride Management:** Drivers can post upcoming trips; passengers can search and book seats.
* **Real-time Database:** Powered by **Supabase** for instant updates on seat availability.
* **Responsive UI:** Fully optimized for both desktop and mobile users.
* **Modern Tech Stack:** Built using Vite for lightning-fast development and deployment.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, TypeScript, Tailwind CSS
* **Build Tool:** Vite
* **Backend/Database:** Supabase (PostgreSQL)
* **Icons:** Lucide React / FontAwesome
